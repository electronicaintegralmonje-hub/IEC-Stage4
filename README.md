**Overview**
This repository contains the theoretical derivation, LaTeX source files, and numerical validation scripts for Stage 4 of the Internal Elasticity Cosmology (IEC) model. 

In this stage, we demonstrate that **Cold Dark Matter (CDM)** emerges naturally as stable topological vortices within the same quantum-coherent elastic field $\phi$ that explains dark energy in Stages 1-3.

**Key Results**
* **Core-Cusp Resolution:** Analytic derivation of a flat density core $r_c \simeq 0.5–2$ kpc.
* **Small-Scale Crisis:** Natural suppression of the matter power spectrum $P(k)$ for $k \gtrsim 5 \, h \, \text{Mpc}^{-1}$.
* **Galaxy Rotation:** Unified fit for SPARC catalog galaxies using discrete winding numbers $n = \pm 1, \pm 2$.
* **Observational Signatures:** Predictions for 21 cm oscillations and CMB B-mode polarization at low multipoles ($\ell < 20$).
